                         Landing Page
                                              
The Aim of project build a multi-section landing page, with a dynamically updating navigational menu based on the amount of content that is added to the page.


Starter code is taken from Udacity [repository]
( https://github.com/udacity/cd0428-landing-page/).

This project MEETS SPECIFICATIONS:


1. The project has 4 sections that have been added to the page.

2. which section is being viewed while scrolling through the page

3. All features are usable across modern desktop, tablet, and phone browsers.


4. It uses JavaScript to dynamically create navbar links based on the content, when a section is in the viewport it shows the active state of that section.

5. the navigation bar is responsive too across all these devices.

6. This project consists of A Responsive layout of the landing page.

7. This project Aim to converting a static web page into a single interactive page.

8. This project is able to be clear which section is being viewed while scrolling through the page.

9. When clicking an item from the navigation menu, the link scroll to the appropriate section.
